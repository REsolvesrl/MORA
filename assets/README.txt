Cartella per gli asset grafici dell'app.

LOGO RESOLVE
------------
Per far comparire il logo in alto nella sidebar e come icona del browser,
salva qui il file del logo con questo nome esatto:

    logo_resolve.png

(PNG, meglio se con sfondo trasparente o lo sfondo navy del brand.)

Finché il file non è presente, l'app funziona lo stesso: mostra
un'icona generica al posto del logo.
